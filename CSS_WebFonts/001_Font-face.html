В предыдущих версиях CSS разработчики были вынуждены использовать только те шрифты,хоторые гарантированно установлены на компьютере пользователя, в CSS3 разработчики могут использовать любые шрифты, которые они захотят. Когда Вы найдете необходимый просто разместите еге на веб-сервере и подключите его с помощью нового CSS3 cвойства @font-face. Подключенный шрифт будет скачан и отображен автоматически при загрузке страницы.

Обратите внимание : Internet Explorer поддерживает шрифты в формате .eot, а осталные браузеры в формате .ttf т.е. для того, чтобы шрифт оттображался корректно во всех браузерах Вы должны подключить его к странице в двух форматах.
